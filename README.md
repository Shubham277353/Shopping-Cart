# DaLimit - Shopping Cart
DaLimit is a affordable shopping corner. Products are fetched from [FakeStore API](https://fakestoreapi.com/)

## Live Demo
[Link here](https://shopping-cart-olive-theta-61.vercel.app/)

## Features
- Three Page SPA : Home, shop and cart with client side routing using react router.
- Error Page : Shows error page in case of unrouted url.
- Responsive design : Auto adjusts to any screen size.
- Add to cart : Persistent and real time updating cart.
- Quantity control : Increment/decrement on shop page itself and will display on cart as well min quantinty 1.
- Order Bill : Real time updating subtotal on quantity increase or product removal.
- Success Page : Page displayed after user clicks checkout , resets cart.

## Technologies Used
- React 19
- React Router
- Vite
- FakeStore API
- TailwindCSS

## How to Run Locally
### Prerequisites
* Node.js 18+
* npm

### Installation
```
git clone https://github.com/Shubham277353/Shopping-Cart.git
cd Shopping-Cart
npm install
```
### Development
```
npm run dev
```
